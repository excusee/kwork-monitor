# Webhook для редактирования откликов (Cloudflare Workers)

Когда отвечаешь (Reply) на сообщение бота со своим отредактированным текстом —
этот воркер находит ссылку на заказ в исходном сообщении и прикрепляет такую же
кнопку «Перейти к заказу» к твоему отредактированному сообщению.

Бесплатно, без карты, работает всегда (не нужно держать комп включённым).

## Установка (один раз)

1. Установи Wrangler (CLI Cloudflare), если ещё нет:
   ```
   npm install -g wrangler
   ```

2. Залогинься (откроется браузer, заведи бесплатный аккаунт Cloudflare, если
   его ещё нет — без карты):
   ```
   wrangler login
   ```

3. Из этой папки (`cloudflare-worker/`) задеплой воркер:
   ```
   wrangler deploy
   ```
   В выводе появится адрес вида `https://kwork-monitor-webhook.<твой-сабдомен>.workers.dev`
   — он понадобится на шаге 5.

4. Добавь токен Telegram-бота как секрет воркера (значение нигде не светится
   в команде, Wrangler спросит его в интерактивном вводе):
   ```
   wrangler secret put TELEGRAM_BOT_TOKEN
   ```

5. Зарегистрируй вебхук у Telegram (выполни сам в своём терминале — подставь
   свой токен бота и адрес воркера из шага 3):
   ```
   curl -F "url=https://kwork-monitor-webhook.<твой-сабдомен>.workers.dev" \
     "https://api.telegram.org/bot<ТВОЙ_ТОКЕН_БОТА>/setWebhook"
   ```
   Ответ должен быть `{"ok":true,"result":true,...}`.

## Проверка

Ответь (Reply) на любое уведомление бота произвольным текстом — должно прийти
сообщение «✏️ Отредактированный вариант» с кнопкой «🔗 Перейти к заказу»,
ведущей на тот же заказ, что был в исходном уведомлении.
